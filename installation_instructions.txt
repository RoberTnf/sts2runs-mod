# STS2 Runs Uploader - Installation Instructions

1) Extract the release zip.
2) Copy the STS2RunsMod folder into your Slay the Spire 2 mods folder.
3) Enable the mod in the in game menu

How to find the installation directory?
Steam -> Slay the Spire 2 -> Right Click -> Manage -> Browse Local Files

Notes:
- The mod creates config.cfg next to the DLL on first run.
- If you update, overwrite files in STS2RunsMod.
- Modding your game causes the game to use a new save file. Use this mod to avoid that and keep your non modded progress: https://www.nexusmods.com/slaythespire2/mods/6

Expected files after install:
- <Slay the Spire 2 install dir>/mods/STS2RunsMod/mod_manifest.json
- <Slay the Spire 2 install dir>/mods/STS2RunsMod/STS2RunsMod.dll
- <Slay the Spire 2 install dir>/mods/STS2RunsMod/installation_instructions.txt (not needed)
